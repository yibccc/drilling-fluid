## 概述

本文档汇总 yibccc-langchain Agent 服务提供的所有 REST API 接口，供前后端开发人员参考。

### 服务信息

| 项目 | 值 |
|------|-----|
| 服务地址 | `http://localhost:8000` |
| API 版本 | v1 |
| 协议 | HTTP/1.1 |
| 响应格式 | JSON / SSE (Server-Sent Events) |
| 认证方式 | API Key (Header: `X-API-Key`) |

### API 模块

| 模块 | 前缀 | 描述 |
|------|------|------|
| 对话模块 | `/api/v1/chat` | LLM 流式对话、工具调用 |
| 诊断模块 | `/api/v1/diagnosis` | 钻井液性能诊断分析 |
| 知识库模块 | `/api/v1/diagnosis/knowledge` | 专家知识库管理 |

---

## 通用说明

### 认证

所有 API 请求（除 `/health`）需在 HTTP Header 中提供 API Key：

```http
X-API-Key: your-api-key
```

### SSE 响应格式

流式接口返回 `text/event-stream`，每行格式：

```http
data: {JSON}

data: {JSON}
```

### 通用错误响应

```json
{
  "detail": "错误描述"
}
```

## 阶段一：流式对话 API

### 1.1 健康检查

**GET** `/health`

无需认证，返回服务状态。

**响应** `200 OK`

```json
{
  "status": "healthy",
  "version": "0.1.0"
}
```

---

### 1.2 流式对话

**POST** `/api/v1/chat/stream`

发送用户消息，接收 SSE 流式响应。

#### 请求头

```http
Content-Type: application/json
X-API-Key: your-api-key
```

#### 请求体

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `message` | string | 是 | 用户消息内容 |
| `session_id` | string | 否 | 已有会话 UUID，首次对话为空 |
| `stream` | boolean | 否 | 是否流式响应，默认 true |

**示例**

```json
{
  "message": "你好，现在几点了？",
  "session_id": null,
  "stream": true
}
```

#### 响应 `200 OK`

**Content-Type**: `text/event-stream`

#### SSE 事件流

```http
data: {"type":"start","session_id":"123e4567-e89b-12d3-a456-426614174000"}

data: {"type":"token","content":"你好"}

data: {"type":"token","content":"！"}

data: {"type":"token","content":"现在是"}

data: {"type":"tool_call","tool_data":{"call_id":"call-abc123","name":"get_current_time","arguments":{"timezone":"Asia/Shanghai"},"status":"calling"}}

data: {"type":"tool_call","tool_data":{"call_id":"call-abc123","name":"get_current_time","arguments":{"timezone":"Asia/Shanghai"},"status":"processing"}}

data: {"type":"tool_result","tool_data":{"call_id":"call-abc123","name":"get_current_time","result":"2026-02-25 15:30:00","status":"result","duration_ms":125}}

data: {"type":"token","content":"2026年2月25日"}

data: {"type":"token","content":"15:30:00"}

data: {"type":"end","content":"stop"}
```

#### SSE 事件类型

| type | 说明 | 数据结构 |
|------|------|----------|
| `start` | 对话开始 | `session_id` |
| `token` | 文本片段 | `content` |
| `tool_call` | 工具调用 | `tool_data.call_id`, `name`, `arguments`, `status` |
| `tool_result` | 工具结果 | `tool_data.call_id`, `name`, `result`, `duration_ms` |
| `end` | 对话结束 | `content` |
| `error` | 错误 | `error_code`, `content` |

#### 工具调用状态

| status | 说明 | 建议图标 |
|--------|------|----------|
| `calling` | 工具正在被调用 | 🔄 |
| `processing` | 工具正在执行 | ⚙️ |
| `result` | 工具执行完成 | ✅ |

## 阶段二：钻井液诊断 API

### 2.1 诊断分析

**POST** `/api/v1/diagnosis/analyze`

提交钻井液预警数据，接收 AI 诊断分析结果。

#### 请求头

```http
Content-Type: application/json
X-API-Key: your-api-key
```

#### 请求体

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `well_id` | string | 是 | 井号 |
| `alert_type` | string | 是 | 预警类型（如 `DENSITY_HIGH`） |
| `alert_triggered_at` | datetime | 是 | 预警触发时间（ISO 8601） |
| `alert_threshold` | object | 是 | 阈值配置 |
| `samples` | array | 是 | 采样数据（1-20条） |
| `context` | object | 否 | 上下文信息（深度、岩性等） |
| `callback_url` | string | 否 | 回调地址 |
| `stream` | boolean | 否 | 是否流式返回，默认 true |

**完整示例**

```json
{
  "well_id": "WELL-001",
  "alert_type": "DENSITY_HIGH",
  "alert_triggered_at": "2026-02-25T10:00:00Z",
  "alert_threshold": {
    "field": "density",
    "condition": "greater_than",
    "threshold": 1.30,
    "current_value": 1.35
  },
  "samples": [
    {
      "id": "SF-001",
      "well_id": "WELL-001",
      "sample_time": "2026-02-25T09:00:00Z",
      "formation": "砂岩",
      "outlet_temp": 58.5,
      "density": 1.22,
      "gel_10s": 3.5,
      "gel_10m": 8.2,
      "rpm_3": 5,
      "rpm_6": 8,
      "rpm_100": 45,
      "rpm_200": 75,
      "rpm_300": 95,
      "rpm_600": 160,
      "plastic_viscosity": 65,
      "yield_point": 15,
      "flow_behavior_index": 0.72,
      "consistency_coefficient": 2.8,
      "apparent_viscosity": 80,
      "yield_plastic_ratio": 0.23
    }
  ],
  "context": {
    "current_depth": 2500.0,
    "formation_type": "砂岩",
    "drilling_phase": "钻进"
  },
  "callback_url": "http://springboot/api/diagnosis/callback",
  "stream": true
}
```

#### 响应 `200 OK`

**Content-Type**: `text/event-stream`

#### SSE 事件流

```http
data: {"type":"start","task_id":"TASK-20260225-123456-AB12CD","well_id":"WELL-001","content":"开始分析井号 WELL-001 的 5 条采样数据"}

data: {"type":"thinking","task_id":"TASK-20260225-123456-AB12CD","content":"正在分析 5 条采样数据...","step":"data_analysis"}

data: {"type":"thinking","task_id":"TASK-20260225-123456-AB12CD","content":"调用工具: analyze_trend({'field': 'density', ...})","step":"tool_call"}

data: {"type":"thinking","task_id":"TASK-20260225-123456-AB12CD","content":"工具 analyze_trend 返回: 趋势上升...","step":"tool_result"}

data: {"type":"thinking","task_id":"TASK-20260225-123456-AB12CD","content":"密度从 1.20 上升到 1.35，需要关注","step":"reasoning"}

data: {"type":"thinking","task_id":"TASK-20260225-123456-AB12CD","content":"调用工具: search_knowledge(...)","step":"tool_call"}

data: {"type":"thinking","task_id":"TASK-20260225-123456-AB12CD","content":"工具 search_knowledge 返回: 密度偏高处置措施...","step":"tool_result"}

data: {"type":"result","task_id":"TASK-20260225-123456-AB12CD","result":{...完整诊断结果...}}

data: {"type":"done","task_id":"TASK-20260225-123456-AB12CD","status":"SUCCESS"}
```

#### SSE 事件类型

| type | step | 说明 |
|------|------|------|
| `start` | - | 分析开始 |
| `thinking` | `data_analysis` | 数据加载和分析准备 |
| `thinking` | `analyzing` | AI 正在分析数据 |
| `thinking` | `tool_call` | 调用诊断工具（analyze_trend, search_knowledge 等） |
| `thinking` | `tool_result` | 工具返回结果 |
| `thinking` | `reasoning` | AI 思考过程（流式输出） |
| `thinking` | `structuring` | 生成结构化诊断结果 |
| `result` | - | 完整结果（包含 diagnosis, trend_analysis, measures, prescription） |
| `done` | - | 分析完成 |
| `error` | - | 错误 |

## 知识库管理 API

### 3.1 创建知识文档

**POST** `/api/v1/diagnosis/knowledge/documents`

创建专家知识文档，自动分块和向量化。

#### 请求体

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `doc_id` | string | 是 | 文档 ID（业务标识） |
| `title` | string | 是 | 文档标题 |
| `category` | string | 是 | 分类（density/viscosity/gel） |
| `subcategory` | string | 否 | 子分类 |
| `content` | string | 是 | 文档内容（Markdown） |
| `metadata` | object | 否 | 元数据 |

**示例**

```json
{
  "doc_id": "DOC-001",
  "title": "密度偏高处置措施",
  "category": "density",
  "subcategory": "high",
  "content": "# 密度偏高处置\n\n当钻井液密度偏高时...",
  "metadata": {
    "author": "专家A",
    "version": "1.0"
  }
}
```

#### 响应 `200 OK`

```json
{
  "doc_id": "DOC-001",
  "status": "created"
}
```

---

### 3.2 语义检索

**POST** `/api/v1/diagnosis/knowledge/search`

基于向量相似度检索知识文档。

#### 请求体

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `query` | string | 是 | 检索查询语句 |
| `category` | string | 否 | 知识分类过滤 |
| `top_k` | integer | 否 | 返回数量，默认 5 |

**示例**

```json
{
  "query": "密度偏高怎么处理",
  "category": "density",
  "top_k": 5
}
```

#### 响应 `200 OK`

```json
{
  "results": [
    {
      "doc_id": "DOC-001",
      "title": "密度偏高处置措施",
      "category": "density",
      "content": "当钻井液密度偏高时...",
      "distance": 0.123
    }
  ]
}
```

## 错误代码

### HTTP 状态码

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | API Key 无效或缺失 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
| 503 | 服务未初始化 |

### 业务错误代码

| 错误代码 | 说明 |
|----------|------|
| `AUTH_FAILED` | API Key 认证失败 |
| `SESSION_NOT_FOUND` | 会话不存在 |
| `LLM_ERROR` | LLM 调用失败 |
| `TOOL_ERROR` | 工具执行失败 |
| `DIAGNOSIS_ERROR` | 诊断分析失败 |
| `RAG_ERROR` | 知识检索失败 |
| `KNOWLEDGE_BASE_ERROR` | 知识库操作失败 |

### 错误响应示例

**SSE 流内错误**

```http
data: {"type":"error","error_code":"LLM_ERROR","content":"DeepSeek API 超时"}
```

**HTTP 错误响应**

```json
{
  "detail": "Invalid API Key"
}
```

## 数据模型

### 对话模块

#### ChatRequest

```typescript
interface ChatRequest {
  message: string              // 用户消息
  session_id?: string          // 会话 ID
  stream?: boolean             // 是否流式
}
```

#### ChatSSEEvent

```typescript
type ChatEventType = 'start' | 'token' | 'tool_call' | 'tool_result' | 'end' | 'error'

interface ChatSSEEvent {
  type: ChatEventType
  session_id?: string
  content?: string
  tool_data?: {
    call_id: string
    name: string
    arguments: Record<string, any>
    status: 'calling' | 'processing' | 'result'
    result?: string
    duration_ms?: number
  }
  error_code?: string
}
```

### 诊断模块

#### DiagnosisRequest

```typescript
interface DiagnosisRequest {
  well_id: string
  alert_type: string
  alert_triggered_at: string  // ISO 8601
  alert_threshold: {
    field: string
    condition: 'greater_than' | 'less_than' | 'equal' | 'between'
    threshold: number
    current_value: number
  }
  samples: DrillingFluidSample[]
  context?: {
    current_depth?: number
    formation_type?: string
    drilling_phase?: string
  }
  callback_url?: string
  stream?: boolean
}
```

#### DiagnosisSSEEvent

```typescript
type DiagnosisEventType =
  | 'start'
  | 'thinking'
  | 'result'
  | 'done'
  | 'error'

type ThinkingStep =
  | 'data_analysis'     // 数据分析准备
  | 'analyzing'         // AI 分析中
  | 'tool_call'         // 调用工具
  | 'tool_result'       // 工具返回结果
  | 'reasoning'         // AI 思考内容
  | 'structuring'       // 生成结构化结果

interface DiagnosisSSEEvent {
  type: DiagnosisEventType
  task_id?: string
  well_id?: string
  content?: string
  step?: ThinkingStep          // thinking 类型时的步骤
  result?: DiagnosisResult     // result 类型时的完整结果
  error_code?: string          // error 类型时的错误代码
}
```
